# 通用在线考试系统

一个基于 Node.js + Express 的全栈在线考试系统，支持多用户、题库管理、自动组卷、自动批改等功能。

## ✨ 功能特性

- 🔐 **用户系统**：注册登录，多用户数据隔离
- 📚 **题库管理**：支持多科目、多套试卷
- 📝 **多种题型**：单选、多选、判断、填空、简答等
- 🎯 **自动组卷**：从题库随机抽题生成试卷
- ✅ **自动批改**：客观题自动评分，主观题手动批改
- 💾 **数据导入导出**：支持批量导入题库
- 📱 **响应式设计**：支持电脑、手机、平板访问

## 🚀 快速开始

### 本地运行

```bash
# 安装依赖
npm install

# 启动服务
npm start
```

访问 http://localhost:3000 即可使用。

### 一键部署到 Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

点击上方按钮，按照提示操作即可一键部署到 Render 平台。

## 📦 技术栈

- **后端**：Node.js + Express
- **前端**：原生 HTML + Tailwind CSS
- **数据存储**：JSON 文件（轻量级，无需数据库）
- **部署**：支持 Render、Railway、Replit 等多种平台

## 📖 使用说明

1. **注册账号**：首次使用请先注册一个账号
2. **导入题库**：登录后在题库管理页面导入你的题库 JSON 文件
3. **开始考试**：选择科目和试卷，开始答题
4. **查看成绩**：提交后自动批改，查看得分和解析

## 📄 题库格式

题库采用 JSON 格式，示例：

```json
{
  "subject": "数据结构",
  "papers": [
    {
      "name": "第一章测试",
      "questions": [
        {
          "type": "single",
          "question": "以下哪个是线性数据结构？",
          "options": ["数组", "树", "图", "堆"],
          "answer": "A",
          "score": 2
        }
      ]
    }
  ]
}
```

导入题库

请使用以下JSON格式导入题库：

{
  "subject": "学科名称",
  "papers": [
    {
      "title": "试卷名称",
      "totalScore": 100,
      "questions": [
        {
          "type": "single", // single单选, fill填空, judge判断, short简答, comprehensive综合
          "question": "题目内容",
          "options": ["A. 选项1", "B. 选项2"], // 仅单选需要
          "answer": "正确答案",
          "score": 2
        }
      ]
    }
  ]
}
  • 包含标准题型标题：单选题、填空题、判断题、简答题、综合题
  • 题目以数字编号开头（1.、2.、3.）
  • 单选题选项以A.、B.、C.、D.开头
  • 答案以"答案："或"参考答案："开头
